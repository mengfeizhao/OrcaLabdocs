# OrcaLab 工具使用文档

欢迎使用 OrcaLab 系列工具的官方文档。本手册将指导你完成从软件安装、设备配置到功能使用的全流程，涵盖 OrcaSim、VR 设备操作及强化学习环境迁移等核心内容。


## 📚 文档导航
通过左侧侧边栏，你可以快速访问以下内容：

- **OrcaLab 使用指南**  
  包含 OrcaSim 安装配置、GameLauncher 部署、VR 设备开发者模式设置及数据合成操作教程。

- **强化学习迁移**  
  详解从 Isaac Gym 到 OrcaGym 的四足机器人强化学习环境迁移方法。


## 🔍 如何使用
1. 首次使用：建议从「OrcaSim 安装与配置指南」开始，完成基础环境搭建。
2. VR 相关操作：需先参考「Pico 设备安装与开发者模式配置」启用设备权限。
3. 强化学习迁移：请确保已完成 Isaac Gym 基础环境部署（详见对应章节前置要求）。


## 📢 注意事项
- 所有工具需在 Windows 10/11 系统下运行，暂不支持 macOS 或 Linux。
- VR 设备配置需使用专用数据线连接电脑，并确保驱动版本兼容（推荐 v5.8.0+）。
- 强化学习迁移过程中若遇到依赖冲突，可参考文档中的「常见问题」章节排查。
